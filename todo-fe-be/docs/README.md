## Project Overview

Todo List Application เป็น project ที่ออกแบบมาเพื่อสาธิตการพัฒนา full-stack application โดยใช้ SDLC methodology แบบ Agile/Iterative ที่ครอบคลุมทุก phase ของการพัฒนา software

**เป้าหมายหลัก:**
- เรียนรู้การพัฒนา full-stack application
- ฝึกการใช้ SDLC ในการจัดการ project
- เข้าใจการทำงานของ database, API, และ frontend integration
- สร้างพื้นฐานสำหรับการพัฒนา project ขนาดใหญ่

**ผู้เรียนเป้าหมาย:**
- นักพัฒนาที่เริ่มต้นเรียนรู้ full-stack development
- ผู้ที่ต้องการเข้าใจ SDLC และการประยุกต์ใช้จริง
- ทีมพัฒนาที่ต้องการ template สำหรับ project structure

## Learning Outcomes

หลังจากศึกษา project นี้แล้ว ผู้เรียนจะสามารถ:

- **เข้าใจ SDLC** และสามารถประยุกต์ใช้กับ project จริง
- **พัฒนา full-stack application** ด้วย modern technology stack
- **ออกแบบ database** และเชื่อมต่อกับ application
- **สร้าง RESTful API** และจัดการ data flow
- **พัฒนา responsive web interface** ด้วย React
- **ใช้ Docker** สำหรับ development และ deployment
- **ทำ testing** และ quality assurance
