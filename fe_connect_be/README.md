- สร้าง form สำหรับกรอกข้อมูล
- กำหนดโครงสร้าง ข้อมูลที่ต้องการเก็บ
- สร้าง route path สำหรับการส่งข้อมูลจาก form ไปยัง server
- เขียน controller สำหรับการรับข้อมูลจาก form และบันทึกข้อมูลลงในฐานข้อมูล

1. สร้าง route api กรณีไม่รับข้อมูลอะไรจากไหนเลย
2. สร้าง route api กรณีรับข้อมูลจาก form
3. สร้าง route api กรณีรับข้อมูลจาก query parameter
4. สร้าง route api กรณีรับข้อมูลจาก path parameter
5. สร้าง route api กรณีรับข้อมูลจาก body (JSON)
6. เเยกอีก 2 กรณี กรณี save data ลง file กับ database

- ดึงข้อมูลจาก database มาเเสดงบน web
