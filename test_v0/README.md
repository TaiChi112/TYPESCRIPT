## Custom Authentication 

**frontend**

- Typescript
- React || Nextjs 

**backend**

- Typescipt
- Express || Elysia

**Objective**
- อยากลองเขียน Authentication ด้วยตัวเอง ที่ผ่านมาใช้ OAuth ทั้งนั้น
  - Sign in
  - Sign up
  - Sign out 
- ฝึกฝนเพิ่มประสบการณ์ในการเขียน Authentication ด้วยตัวเอง

**Problem**
- ยังไม่มั่นใจใน skills การเขียน Authentication

**Solution**
- ลองฝึกเขียน webiste ที่ต้องใช้ Authentication เเบบง่ายๆ

**Requirement**
- Website ที่มีการ Authentication
- มีการ Sign in, Sign up, Sign out
- มีการเก็บข้อมูลผู้ใช้ในฐานข้อมูล
- มี home page , sign in page, sign up page, sign out page
